# Advanced Weather & AQI Web App (Starter)

## Overview
Starter scaffold for an advanced Weather & Air Quality web app using Flask (Python) for backend and HTML/CSS/JS for frontend.
It includes placeholders for API integration (OpenWeatherMap / IQAir), favorites, and a polished UI scaffold.

## What's included
- `app.py` — Flask app with example routes
- `api_handlers.py` — API wrapper placeholders for weather & AQI
- `utils.py` — helper functions (AQI colour mapping, wind direction, unit toggles)
- `db.py` — simple SQLite helper for saved locations
- `templates/index.html` — Main UI
- `static/css/style.css` — Tailwind-like utility + custom styles (no build step)
- `static/js/main.js` — Frontend logic: search, fetch, render
- `requirements.txt` — Python dependencies

## Quick start (local)
1. Create a virtualenv and install requirements:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
2. Set API keys as environment variables (or edit `api_handlers.py`):
   - `OPENWEATHER_API_KEY`
   - `IQAIR_API_KEY` (optional)
3. Run the Flask app:
   ```bash
   export FLASK_APP=app.py
   export FLASK_ENV=development
   flask run
   ```
4. Open http://127.0.0.1:5000

## Notes
- This scaffold intentionally keeps third-party integrations minimal & uses placeholders so you can plug your keys.
- The frontend uses vanilla JS and Chart.js (CDN) for charts.
